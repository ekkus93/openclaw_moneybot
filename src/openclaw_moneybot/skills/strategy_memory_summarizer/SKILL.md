# strategy_memory_summarizer

## Purpose

Convert experiment outcomes into reusable structured lessons grounded in ledgered facts.

## Inputs

- experiment review metrics
- payout reconciliation outcome
- optional counterparty-risk context

## Outputs

- lesson categories
- what worked / what failed
- heuristics to keep / avoid
- tentative hypotheses when evidence conflicts

## Fail-closed behavior

- contradictory outcomes stay tentative
- summaries never replace the underlying historical record

## Non-goals

- speculative narrative memory
- silently overwriting prior lessons
