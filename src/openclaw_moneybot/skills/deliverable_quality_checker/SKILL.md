# deliverable_quality_checker

## Purpose

Verify that required deliverables and proof artifacts are present before submission or review completion.

## Inputs

- submission package requirements
- prepared field values and artifacts

## Outputs

- pass/fail/review status
- missing and invalid item lists
- passed checks

## Fail-closed behavior

- missing required artifacts fail the check
- placeholder text never counts as a valid deliverable

## Non-goals

- generating deliverables
- auto-submitting them
